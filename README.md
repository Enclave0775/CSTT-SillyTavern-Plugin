# CSTT-SillyTavern-Plugin - SillyTavern 專用全方位繁簡轉換工具

CSTT-SillyTavern-Plugin 是一個為 SillyTavern 設計的第三方擴充功能，主要功能是讓使用者能夠在多種中文變體（簡體、繁體、臺灣正體、香港繁體等）之間轉換角色卡（`.png`）、世界書（`.json`）、預設（`.json`）檔案。

本工具能穩定地處理儲存在 PNG 檔案中的元數據（metadata），確保在轉換文字的同時不會損壞圖片檔案。

## 主要功能

- **全方位中文轉換：** 支援 OpenCC 所有主要轉換模式，包含：
    - 簡體 <-> 繁體 (標準)
    - 簡體 <-> 臺灣正體 (含/不含慣用語)
    - 簡體 <-> 香港繁體
    - 繁體 <-> 臺灣正體/香港繁體
- **支援多種檔案類型：**
    - **角色卡 (PNG):** 智慧型解析 PNG 區塊（`tEXt`, `zTXt`, `iTXt`），這些區塊通常儲存著角色資料。自動偵測並處理元數據中的原始 JSON 或 Base64 編碼的 JSON。在修改後，會以正確的區塊長度和 CRC 校驗和重建 PNG 檔案，以防止檔案損毀。
    - **世界書 (JSON):** 支援轉換世界書檔案。
    - **預設 (JSON):** 支援轉換預設檔案。
    - **正規表達式 (JSON):** 支援轉換正規表達式腳本並匯入。
- **AI 回覆即時轉換：** 自動攔截 AI 生成的回覆，並將其轉換為您偏好的語言變體（如轉為臺灣繁體）。
- **友善的使用者介面：** 在 SillyTavern 的擴充功能面板中提供了一個簡單的介面，方便使用者上傳檔案並選擇轉換模式。
- **自動匯入：** （可選功能）轉換後的檔案可以透過模擬拖放的方式自動匯入回 SillyTavern，簡化了操作流程。
    - **注意：** 預設檔案與正規表達式的匯入功能需要 **[酒館助手](https://gitlab.com/novi028/JS-Slash-Runner)** 插件支援。

## 安裝方式

1.  開啟 SillyTavern。
2.  點擊頂部工具列中的「擴充功能 (Extensions)」按鈕。
3.  點擊「安裝擴充功能 (Install Extension)」。
4.  將此 URL 複製到輸入欄位中： `https://github.com/Enclave0775/CSTT-SillyTavern-Plugin`
5.  點擊「僅為我安裝 (Install just for me)」或「為所有使用者安裝 (Install for all users)」。

## 使用教學

### 檔案轉換

1.  在 SillyTavern 中，前往擴充功能的設定頁面。
2.  找到 **CSTT Converter** 面板。
3.  點擊 **「選擇檔案 (Select Files)」** 按鈕，選擇一個或多個您想要轉換的 `.png` 或 `.json` 檔案。
4.  從下拉選單中選擇您需要的 **轉換模式 (Conversion Mode)**，例如：
    - **簡體 -> 臺灣正體 (含慣用語)**
    - **繁體 -> 簡體 (標準)**
    - **簡體 -> 香港繁體**
    - ...等 10 種模式。
5.  （可選）如果您希望轉換後的檔案能自動載入回 SillyTavern，請勾選 **「自動匯入 (Auto Import)」** 核取方塊。如果未勾選，檔案將會直接下載到您的裝置。
6.  點擊 **「轉換 (Convert)」** 按鈕。
7.  下方的日誌區域將會顯示轉換過程的狀態。

### AI 回覆即時轉換

1.  在擴充功能設定頁面中，找到 **AI 回覆即時轉換** 區塊。
2.  勾選 **「啟用 AI 回覆攔截轉換」**。
3.  從下拉選單選擇您希望 AI 回覆轉換成的目標語言模式。
    4.  設定完成後，AI 生成的每一條新訊息都會在生成結束後自動轉換為您指定的語言。

### 外掛字典 (External Dictionary)

此功能允許您載入自定義的字典檔來覆蓋或擴充內建的轉換規則。

1.  在擴充功能設定頁面中，找到 **外掛字典管理 (Dictionary Manager)** 區塊。
2.  點擊 **「新增/匯入字典...」** 按鈕，選擇您的字典檔案。
    *   **支援格式**：`.txt` (每行 `原文 替換文`) 或 `.json` (二維陣列或物件)。
3.  匯入後，字典會顯示在列表中。您可以勾選前方的方塊來 **啟用/停用** 特定字典。
4.  已啟用的外掛字典會在檔案轉換與 AI 回覆轉換時**優先**生效。
5.  點擊右側的 **下載** 圖示可將字典備份回電腦，點擊 **垃圾桶** 圖示可刪除。

### 字典工具 (Dictionary Tools)

提供字典格式轉換的輔助工具。

1.  在 **字典工具** 區塊，點擊 **「選擇字典檔案...」** 選擇您的原始字典檔（如 `.txt`）。
2.  若勾選 **「轉換後自動加入外掛字典列表」**：
    *   點擊 **「轉為 JSON」** 後，字典會被自動轉換並掛載到上方的「外掛字典管理」中，方便直接使用，且不會下載檔案。
3.  若**取消勾選**：
    *   點擊 **「轉為 JSON」** 後，會將字典轉換為 OpenCC 相容的 JSON 格式並下載到您的電腦。

## 技術細節

本擴充功能使用了數個強大的函式庫來實現其功能：

- **[OpenCC-JS](https://github.com/nk2028/opencc-js):** 核心的繁簡中文轉換邏輯。
- **[Pako](https://github.com/nodeca/pako):** 用於解壓縮在 `zTXt` 和 `iTXt` PNG 區塊中找到的壓縮資料。
- **CRC32:** 用於在修改後重新計算每個 PNG 區塊的循環冗餘校驗（CRC）值，這對於維持 PNG 檔案的完整性至關重要。
